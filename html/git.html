Git

  • git init-cоздать пустой репозиторий Git или вновь инициализировать существующий.
  • git add -добавить отдельный файл в область подготовленных файлов.
  • git status-просмотреть статус нужного репозитория.
  • git commit-с помощью этой команды при создании коммита в репозитории можно добавить однострочное сообщение.
  • git reset- восстановить подготовленный файл рабочего дерева.
